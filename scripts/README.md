# Scripts de Automação

Esta pasta contém os scripts utilitários para o projeto AutomacaoLotes.

## Arquivos

- **criar_executavel.bat** - Script para criar o executável do projeto
- **gerar_exe.bat** - Script para gerar o arquivo executável
- **gerar_exe.sh** - Versão Shell Script para ambientes Unix/Linux
- **desbloquear_arquivo.bat** - Script para desbloquear arquivos bloqueados pelo Windows
- **desbloquear_arquivo.ps1** - Versão PowerShell para desbloquear arquivos

## Uso

Execute os scripts a partir da raiz do projeto ou ajuste os caminhos conforme necessário.
